Whats in this repository? batch-represent: Generate representations from a batch of images. Example directory structure. demos/web: Real-time web demo. demos/compare.py: Demo to compare two images. demos/vis-outputs.lua: Demo to visualize the networks outputs. demos/classifier.py: Demo to train and use classifiers. demos/classifier_webcam.py: Demo to use a trained classifier on a webcam stream. evaluation: LFW accuracy evaluation scripts. openface: Python library code. models: Model directory for openface and 3rd party libraries. tests: Tests for scripts and library code, including neural network training. training: Scripts to train new OpenFace neural network models. util: Utility scripts.
